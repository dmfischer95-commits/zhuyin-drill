# ⚡ Zhuyin Drill (Bopomofo)
> **The Playstore was mid, so I vibe-coded a better solution.**

## 💡 The Vision
I needed a tool to master the Taiwanese phonetic system (Zhuyin/Bopomofo) the way I learned Japanese Kana 10 years ago: **minimalist, zero-latency, and brutal.** Most existing apps are bloated with animations and "gamification" that slows down the actual learning process. I wanted a drill that respects my time.

## 🛠️ Vibe Coding Mode
This project was built in **less than 48 hours** using **Gemini AI** and **Claude**. 
- **Philosophy:** Execution over perfection. 
- **Method:** AI-accelerated development with zero double-checking. 
- **Result:** A fully functional, production-ready PWA that just works.

## 🚀 Core Features
- **Zero-Latency Audio:** Powered by a custom `AudioContext` engine. No lag between click and sound.
- **Smart Priority Queue:** Detects your weak spots and forces you to repeat them until they stick (Kahneman's System 1 training).
- **100% Offline-First:** PWA with Service Worker caching. Perfect for the subway or flight mode.
- **Privacy-First:** No tracking, no ads, no backend. Your stats stay in your `localStorage`.
